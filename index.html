Computer Vision

In this project, the goal was to develop an autonomous bot for Counter-Strike: Global Offensive (CS:GO) using the YOLOv8 engine and Python, without relying on internal game data. By using AI and machine learning techniques for real-time visual data interpretation and decision-making, this bot demonstrates an innovative approach to game automation. It skillfully navigates the game environment, identifies opponents, and interacts within the game purely through visual analysis, showcasing the potential of AI in transforming gaming experiences.

Fraud Detection Model

In an engaging project from my Machine Learning class, I aimed to simulate authentic bank transaction data, highlighting the challenge of detecting fraud where only 0.05% of transactions were fraudulent. Utilizing advanced machine learning techniques, I developed a model capable of discerning fraudulent transactions with approximately 78% accuracy. This project not only showcased the intricacies of handling imbalanced datasets but also demonstrated the potential of machine learning in enhancing financial security.
